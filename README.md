**G-iHUB** is a comprehensive platform focused on fostering a pan-African startup ecosystem. Its mission is to empower entrepreneurs and teams through a structured, mentor-led journey from initial ideation to large-scale market entry.

The platform is closely integrated with **StartupVerse**, a central digital hub for managing mentors, curricula, and vital resources for founders.

## 🚀 Features & Sections

The website features a clean, energetic, and modern design, utilizing bright orange (`#FF4D00`) for action elements and dark navy (`#0B0F19`) to balance professionalism with dynamic tech-startup energy.

Key sections of the platform include:

- **The Founder Journey**: A core four-stage framework for development:
  1. **Ideation**: Validating ideas through "Founder Challenges" and "Innovation Clinics" with expert support.
  2. **Execution**: Building products on the StartupVerse platform guided by mentors.
  3. **Incubation**: Access to physical workspaces, operational support, and investor introductions.
  4. **Scale**: Entering Pan-African markets via partner networks and capital connections.
- **Services**: Detailed overviews of support offered across the lifecycle of a startup.
- **Management & Team**: An introduction to the leadership and domain experts driving the vision.
- **History**: A timeline of the organization's ecosystem milestones and evolution.
- **News & Insights**: ("Here's What We're Talking About") A blog or news hub highlighting ecosystem updates and trends.

## 🎨 Visual Identity

- **Primary Colors**: Bright Orange (`#FF4D00`) for emphasis and Dark Navy (`#0B0F19`) for text and contrast.
- **Typography**: Clean, readable sans-serif (such as Inter or Roboto) to maintain a modern aesthetic.
- **Design Archetype**: Energetic, structured, and modern, balancing financial incubation with a dynamic community focus.

## 🛠️ Tech Stack

This project is bootstrapped and built with modern web technologies for high performance and rapid development:

- **React 19**: A powerful UI library for writing maintainable and scalable components.
- **Vite**: A lightning-fast build tool and development server.
- **TypeScript**: Ensuring strongly typed code out of the box.

## 📦 Getting Started

First, ensure you have Node.js installed, then install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Your app will be available locally (usually at `http://localhost:5173`).

## 🏗️ Build & Preview

To build the project for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
